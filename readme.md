
# ropin-usdviewer

roah-apine エンジンによる usdviewer アプリです。

**RO**ah a**PIN**e USDVIEWER、「ローピン　ユーエスディービューワー」です。

apine = **AP**p eng**INE**、「アパイン」です。

なので **RO**ah a**P**p eng**IN**e で ropin です。どうでもいいですね。

## やりたいこと

### HDR で usdview したい

ひとまず Storm を HDR でレンダリングして、 HDR の画面で見たいよね。

### roah-apine エンジンをデバッグしたい

とりあえず何かアプリを作りながら、 roah-apine エンジンをデバッグしたい、というのが割と第一の目的。

## やらないこと

機能的には usdview の超絶劣化版になります。

シーンを細かく見たい、アトリビュートを確認したい用途には向かないでしょう。
